EventFlow

EventFlow is a lightweight, full-stack event management solution designed to bridge the gap between event organizers and attendees. Developed as a fast-paced prototype, this project focuses on a seamless user experience and a robust backend for real-time seat tracking.

The Problem
During large-scale events or hackathons, organizers often struggle with:

Fragmented Data: Tracking attendee details (contact info, food preferences) across different sheets.

Overbooking: Managing real-time seat availability manually.

Information Silos: Attendees not having access to detailed speaker or venue info before booking.

Our Solution
We built a centralized platform that handles the entire lifecycle of an event:

For Admins: A specialized dashboard to publish events with 14 data points (Venue, Speakers, Guests, Pricing, etc.) and a live "Attendee List" for logisitics planning.

For Users: A responsive discovery portal with detailed registration modals that capture critical data (Contact info & Dietary preferences) at the point of booking.
